# Api-university
Api desarrollada en Java con Spring

## Características
* Operaciones CRUD (create, read, update, delete)
* Registro de usuario y autentificación con Jason Web Token
* Autorizacion basada en roles con Spring Security

## Tecnologías
* Java 17
* Spring Boot 3
* Spring Security 6
* JSON Web Token (JWT)
* PostgreSQL Database
* BCrypt
* Maven
